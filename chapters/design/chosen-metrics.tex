\section{Výběr metrik}
S ohledem na různé druhy činností prováděných během vývoje softwaru si na rozdíl například od \cite{measuring-the-individual-performance-of-a-software-development-team} tato práce nedává za cíl vybrat nebo vytvořit metriku, která by se snažila pomocí jednoho čísla popsat celkovou výkonnost vývojářů. Dle SPACE frameworku není ani možné produktivitu vývojářů vysvětlovat a měřit pomocí jedné dimenze (metriky). \cite{space} Z tohoto důvodu bylo vybráno několik z dříve představených metrik z oblasti code review a úpravy zdrojového kódu, které lze pomocí GraphQL API získat jak z nástroje GitHub, tak z nástroje GitLab.
